\[ H|x(r, t)\rangle = i\hbar \frac{\delta}{\delta t}|x(r,t)\rangle \]

\[ (1 + \frac{it}{2} H)|x(r,t)\rangle^{n+1} = (1 - \frac{it}{2} H)|x(r,t)\rangle^{n} \]

LinearHamiltonianSolverBasic linear Schr√∂dinger equation solver which solves the Crank Nicolson algorithm. The LinearHamiltonianSolver solves the Schr√∂dinger equation with the form

\[ (\frac{P^2}{2m} + V(r))|x(r, t)\rangle = i\hbar \frac{\delta}{\delta t}|x(r,t)\rangle \]

with the \(V(r)\) potential at the position \(r\). The default solver used a left and a right matrix to solve the equation with:

\[ (1 + \frac{it}{2h} H)|x(r,t)\rangle^{n+1} = (1 - \frac{it}{2h} H)|x(r,t)\rangle^{n} \]

with

\[ Left := 1 + \frac{i\Delta t}{2h} H \]

and

\[ Right := 1 - \frac{i\Delta t}{2h} H \]
