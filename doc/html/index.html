Crank Nicolson

Introduction

Crank Nicolson solver for one dimensional waves. This program solves the timedependent Schroedinger equation for one dimensional waves. The only parameter which is nessesary is the simulation file. This simulation file contains the data and a path to a script to run a simulation.
